### 安装依赖

`npm install`

### 启动

`npm run watch`

### 执行 `npm link`

此时就可以使用 `eos` 命令了。

- `eos init vue-template myVue`
- `eos config get`
- `eos config set type orgs`
- `eos config set registry vuejs-templates`

- `eos config set type users`
- `eos config set registry YvetteLau`

### 发布

开发完成后，即可发布至 npm.
